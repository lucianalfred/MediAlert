# MediAlert
MediAlert App 

# The Problem

This project consists of a web application developed to help people who have difficulty remembering the correct times to take their medication. The solution is especially useful for the elderly, patients undergoing prolonged treatments, and anyone who needs assistance with medication administration.

Through the application, the user can register their prescription, add prescribed medications, set dosing times, and configure automatic reminders. When it's time to take the medication, the system sends a notification via WhatsApp, ensuring the user is alerted in a timely manner.

In addition, the application allows users to mark each medication as "taken," enabling treatment tracking and recording of dosing history. In this way, the solution contributes to greater medication adherence, reducing forgetfulness and promoting better healthcare.
